## ğŸ› Troubleshooting

### Widget shows "Loading..." forever
1. Check internet connection
2. Disable ad-blockers
3. Refresh page (Ctrl+R / Cmd+R)

### Widgets disappear when I reload
1. Check browser allows localStorage
2. Don't use incognito/private mode
3. Check browser console for errors

### Dashboard is slow/laggy
1. Close unused widgets (click X)
2. Limit to 4-6 active widgets
3. Close other browser tabs
4. Restart browser

### Theme doesn't match
1. Toggle theme (sun/moon in sidebar)
2. Reload page
3. Remove and re-add widgets
